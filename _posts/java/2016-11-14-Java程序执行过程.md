#Java中的数据类型#
##基本数据类型##
1. 整数 ： 固定的表数范围和字段长度，不受操作系统的影响
	* byte  ：  1字节   -128~127；
	* short  :  2字节   ；
	* int : 4字节；
	* long : 8字节；
2. 浮点型：
	* float : 4字节；
	* double : 8字节；

Java浮点型常量认为是double类型，因此要声明一个变量为float，则需要在数字后面加f 或F；
java整型变量默认是int变量；因此要声明一个变量为long 则需要在数字后面添加l或L，推荐都使用大写
